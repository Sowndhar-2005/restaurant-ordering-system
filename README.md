# Restaurant Ordering System

A full-stack web application for managing restaurant orders and menu management. This project demonstrates modern web development practices with Angular frontend and Java backend.

## Features

### Customer Features
- Browse restaurant menu with categories
- Search and filter menu items
- Add items to cart
- Place and track orders

### Admin Features
- Menu management (add, edit, delete items)
- Order management and status tracking

## Technology Stack

**Frontend**: Angular, TypeScript, CSS
**Backend**: Java, Spring Boot
**Database**: MySQL/PostgreSQL

## Project Structure

```
restaurant-ordering-system/
├── backend-java/     # Spring Boot backend
└── frontend-angular/ # Angular frontend
```

## Installation

### Backend
```bash
cd backend-java
mvn clean install
mvn spring-boot:run
```

### Frontend
```bash
cd frontend-angular
npm install
ng serve
```

RESTful APIs for menu, orders, and user management.
